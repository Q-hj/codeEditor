# codeEditor

## 自定义编程语言

### 代码高亮

> 可通过 languages.setMonarchTokensProvider 为对应语言进行代码高亮配置

### 语法提示

> 可通过 languages.registerCompletionItemProvider 为对应语言进行代码提示配置。

## 功能

- 编辑器

  - 主题切换
  - 语言切换
  - 字体大小调整
  - 分屏（待做）
  - 跳转到指定行数（待做）

- 编程语言
  - 代码高亮
  - 代码提示
  - 代码格式化（待做）
  - 语法校验（待做）
  - 悬浮提示（待做）
